# 选股助手

一个纯静态的 A 股观察和选股辅助网页，设计风格沿用现有 tracker。

## 功能

- 股票增删改查，优先同步到 Supabase 在线数据库，失败时保存在浏览器 localStorage。
- 通过公开行情接口刷新股票现价、涨跌幅、日内高低、成交额等基础信息。
- 设置价格区间，默认人民币 70 元以内。
- 固定默认选股时间 14:35。页面打开时，到点会自动生成一次本地候选。
- 设置买入量，默认 1 手。
- 无 OpenAI API 模式：网页生成可复制给 ChatGPT 或 Codex 的提示词，由用户在订阅版产品里粘贴分析。

## 重要说明

个人 ChatGPT 订阅不能被网页直接调用。要在网页中自动调用模型，需要 OpenAI API key 或后端服务。本项目因此采用“复制提示词”的协作方式，不会保存或传输你的 OpenAI 账号信息。

本工具只做信息整理和候选筛选，不构成投资建议。

## Supabase

先在 Supabase SQL Editor 运行 `supabase-schema.sql`。表建好后，网页会自动连接 `picker_stocks` 和 `picker_settings`，实现多设备共享。
